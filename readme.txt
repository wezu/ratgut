Usage: ratgut.py [-res] [-edit] [-preset]

Options:
-res    resolution of the created maps (default 512)
-edit   show an interactive gui for editing the heightmap
-preset define a preset to generate a new map, the presets are:
        moutains
        hills
        dunes
        ridge
        foothills
        plains
        spikes
        canyon
        
You will need the Panda3D SDK to make this work.
